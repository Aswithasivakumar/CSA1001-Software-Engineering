# Software Engineering Learning Portal 💻

An interactive web application developed using Streamlit for Software Engineering case studies and assignment demonstration.

## Features
- 📚 Five Software Engineering case studies
- ✅ Scrum Framework Analysis
- ✅ Smart Parking Feasibility Study
- ✅ Lean + XP Methodology
- ✅ SAFe Framework for Large Projects
- ✅ Ethical AI Evaluation
- 📝 Mini Quiz
- 🎓 Completion Certificate

## Technologies Used
- Python
- Streamlit
- Pandas
